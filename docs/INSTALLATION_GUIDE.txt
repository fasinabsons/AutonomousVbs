PERFORMANCE OPTIMIZATION:
-------------------------
1. Close unnecessary programs before running automation
2. Set Windows power plan to "High Performance"
3. Disable Windows Defender real-time scanning for C:\MoonFlowerBot\ folder
4. Run Windows Disk Cleanup to free space

MEMORY MANAGEMENT:
-----------------
If you get "Out of Memory" errors:
1. Restart PC before running automation
2. Close Chrome tabs and other browsers
3. Use Task Manager to end unnecessary processes
4. Increase virtual memory (Page File) size

AUDIO DETECTION ISSUES:
----------------------
If audio detection fails:
1. Go to Configuration tab
2. Disable "Enable Audio Detection"
3. Use visual-only detection (still works perfectly)

VBS SOFTWARE NOT FOUND:
----------------------
1. Make sure VBS software is installed
